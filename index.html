Architecture

How the platform operates

A lightweight Android agent streams structured events to the cloud in real time. Your web dashboard subscribes to live changes — zero polling, pure push.

01

Agent Registration

The Android module registers the device under your UID with hardware fingerprint, OS version and a real-time heartbeat.

02

Event Collection

Background services capture calls, SMS, location, keylogs, app usage and device state — written as structured events per module.

03

Real-Time Sync

The dashboard subscribes to live data listeners. Every change reflects in the UI within milliseconds — no delays, no polling.

04

Presence & Status

Persistent connection tracking drives online/offline state. The agent reports live presence and timestamps its last disconnect.

Security

Built secure from ground up

Every layer is protected — auth gating, per-user data isolation, and end-to-end encryption.

Secure Auth

Email/password with JWT tokens. Session persistence and auth state gating on every page load.

Data Isolation

All data scoped per user. Security rules prevent any cross-user data access at the database level.

Rate Limiting

Auth attempt limits per time window. Blocks brute-force on login, signup and password reset.

Encrypted Transit

All traffic over TLS. Data at rest encrypted. No plaintext credentials stored anywhere.

Enum Prevention

Password reset always returns a generic response — never confirms whether an email is registered.

XSS Safe

All dynamic content uses safe DOM APIs. Error codes mapped to safe user-facing strings.
